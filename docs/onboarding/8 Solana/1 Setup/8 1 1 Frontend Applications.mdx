You can build front-end applications by using our [React](/react) and [TypeScript](/typescript) SDKs!

Front-end applications are best-suited when you want users to connect their wallets to your web application and interact with your programs using their wallet.

## Create A New Project

The easiest way to get started is by using the [CLI](https://github.com/web3sdks/web3sdks-cli).

```bash
npx web3sdks@latest create app --solana --ts
```

You're project is ready to start interacting with your programs, tokens and NFTs on Solana.

## Integrate Into Existing Projects

If you have an **existing** project you want to integrate web3sdks into, install the web3sdks SDKs:

```bash npm2yarn
npm install @web3sdks/sdk @web3sdks/react @solana/wallet-adapter-wallets @solana/wallet-adapter-react
```

Then wrap your application in the `Web3sdksProvider` to get started.

```jsx
import { Web3sdksProvider } from "@web3sdks/react/solana";

// Change the network to the one you want to use: "mainnet-beta", "testnet", "devnet", "localhost" or your own RPC endpoint
const desiredNetwork = "devnet";

export const MyApp = () => {
  return (
    <Web3sdksProvider network={desiredNetwork}>
      {/* Your App Goes Here */}
    </Web3sdksProvider>
  );
};
```

Any page wrapped within the `Web3sdksProvider` has access to the features of the React SDK, such as:

- [`useWallet`](https://solana-labs.github.io/wallet-adapter/#usage) - Get the current wallet and its state
- [`useSDK`](/react/solana/react.usesdk) - Get the current SDK instance
- [`useProgram`](/react/solana/react.useprogram) - Get any deployed program instance

This setup is compatible with the [Solana Wallet Adapter](https://github.com/solana-labs/wallet-adapter) library. You can use any of the supported wallets.

### Existing Wallet Connection Setups

If you already have a connection and wallet setup using [Solana's web3js](https://solana-labs.github.io/solana-web3.js/),
you can use the `Web3sdksSDKProvider` to access the SDK functions without any of the wallet management functionality.

```jsx
import { Web3sdksSDKProvider } from "@web3sdks/react/solana";
import { useWallet } from "@solana/wallet-adapter-react";

export const MyApp({ Component, pageProps }) {
  const wallet = useWallet();

  return (
    <Web3sdksSDKProvider network={"devnet"} wallet={wallet}>
      {/* Your App Goes Here */}
    </Web3sdksSDKProvider>
  );
}
```
