### Before You Begin

We recommend you have the following tools installed before you begin:

- [Node.js and npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm): To install our SDK into your project.
- [Yarn](https://classic.yarnpkg.com/en/docs/install/#mac-stable): For executing scripts in our React app.
- [Git](https://git-scm.com/downloads): For using the `create` command in our CLI.
