# ERC1155 Standard

You can utilize these features of the SDK if your contract implements the [ERC1155](/contractkit/interfaces/erc1155) standard.

## View NFT Balance

Get a wallet's NFT balance (number of NFTs in this contract owned by the wallet) for a specific token ID.

<Web3sdksFeatureCodeSnippet
  featureName={"ERC1155"}
  languageFunctionMapping={{
    javascript: "balanceOf",
    react: "useNFTBalance",
  }}
/>

## Get NFT Metadata

Read the URI this NFT points to and fetch that data automatically.

If the metadata is stored on IPFS, this function uses our IPFS gateway (or the one you specify) to read the data.

<Web3sdksFeatureCodeSnippet
  featureName={"ERC1155"}
  languageFunctionMapping={{
    javascript: "get",
    react: "useNFT",
  }}
/>

## Transfer NFTs

Transfer an NFT from the connected wallet to another wallet.

<Web3sdksFeatureCodeSnippet
  featureName={"ERC1155"}
  languageFunctionMapping={{
    javascript: "transfer",
    react: "useTransferNFT",
  }}
/>

## Airdrop NFTs

Transfer NFTs to a list of wallets in one transaction.

<Web3sdksFeatureCodeSnippet
  featureName={"ERC1155"}
  languageFunctionMapping={{

    react: "useAirdropNFT",
  }}
