# Non-Transferable Tokens

Configure tokens in your smart contract to be non-transferable (or "soulbound").

<div className="row" style={{ marginBottom: 24 }}>
  <div className="col col--12" style={{ padding: 8 }}>

To make tokens in your contract non-transferable, set the [transfer role](/sdk/advanced-features/permission-controls) to be empty.

```javascript
const rolesAndMembers = await contract.roles.getAll();

const updatedRoles = {
  ...rolesAndMembers,
  transfer: [],
};

await contract.roles.setAll(updatedRoles);
```
