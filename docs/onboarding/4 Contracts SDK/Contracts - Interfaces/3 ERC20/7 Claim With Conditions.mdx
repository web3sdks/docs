# ERC20ClaimableWithConditions

Allow other wallets to claim/mint tokens from the contract under the criteria of claim conditions by
implementing `ERC20` and `DropSinglePhase`.

## Unlocked Features

By implementing the `ERC20` standard, you unlock the following features in the SDK and dashboard:

| SDK Feature                                                                                    | Description                                                                       |
| ---------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| [Claim Tokens](/sdk/interacting-with-contracts/erc721lazymintable#claim-a-lazy-minted-nft)     | Claim (mint) a lazy-minted NFT to a wallet address.                               |
| [Claim Conditions](/sdk/interacting-with-contracts/erc721Claimable#configure-claim-conditions) | Mints a quantity of lazy-minted NFTs from the drop to a specified wallet address. |

    }
