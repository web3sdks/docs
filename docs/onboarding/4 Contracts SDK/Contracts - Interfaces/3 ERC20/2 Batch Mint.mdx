## Unlocked Features

By implementing the `ERC20` standard, you unlock the following features in the SDK and dashboard:

| SDK Feature                                                                                               | Description                                          |
| --------------------------------------------------------------------------------------------------------- | ---------------------------------------------------- |
| [Batch Mint Tokens](/sdk/interacting-with-contracts/erc20/erc20batchmintable#mint-tokens-to-many-wallets) | Mint tokens to many wallets in a single transaction. |

## Implementing It Yourself

This section is meant for advanced users who want to write the functionality from scratch.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@web3sdks/contracts/openzeppelin-presets/token/ERC20/ERC20.sol";
import "@web3sdks/contracts/extension/interface/IMintableERC20.sol";
import "@web3sdks/contracts/extension/Multicall.sol";

contract Contract is ERC20, IMintableERC20, Multicall {
    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC20(
            _name,
            _symbol
        )
    {}

    function mintTo(address to, uint256 amount) external override {
        // Your custom implementation here
    }
}
```
