## Unlocked Features

By implementing the ERC1155Mintable contract, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                                                                                 | Description                                                              |
| ------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| [Lazy Mint Batch](/sdk/interacting-with-contracts/erc1155lazymintable#create-a-batch-of-nfts-to-be-claimed-in-the-future) | Pass in an array of NFT metadata and lazy-mint them for others to claim. |

## Implementing It Yourself

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@web3sdks/contracts/eip/ERC1155.sol";
import "@web3sdks/contracts/extension/LazyMint.sol";

contract Contract is ERC721A, LazyMint {
    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC721A(
            _name,
            _symbol
        )
    {}

    function _canLazyMint() internal view override returns (bool) {
        // Your custom implementation here
    }
}
```
