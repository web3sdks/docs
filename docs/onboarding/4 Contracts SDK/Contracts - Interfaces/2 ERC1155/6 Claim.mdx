## Unlocked Features

By implementing `ERC1155Claimable`, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                                                | Description                                         |
| ---------------------------------------------------------------------------------------- | --------------------------------------------------- |
| [Claim NFT](/sdk/interacting-with-contracts/erc1155lazymintable#claim-a-lazy-minted-nft) | Claim (mint) a lazy-minted NFT to a wallet address. |
