## Unlocked Features

By implementing `ERC1155Claimable`, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                               | Description                                                                   |
| ----------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| [Delayed Reveals](/sdk/interacting-with-contracts/erc1155delayedreveal) | Pass in an array of NFT metadata to encrypt it and reveal it at a later time. |
