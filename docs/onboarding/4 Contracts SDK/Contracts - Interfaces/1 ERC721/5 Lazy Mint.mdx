# ERC721LazyMintable

Implement `ERC721A` and `LazyMint` to enable batch lazy minting of NFTs in your smart contract
and to let other people claim (mint) the lazy minted NFTs.

## Unlocked Features

By implementing `ERC721Mintable`, you unlock the following features in the SDK and dashboard:

| SDK Usage                                                                                                                       | Description                                                              |
| ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| [Lazy Mint Batch](/sdk/interacting-with-contracts/erc721lazymintable#create-a-batch-of-unique-nfts-to-be-claimed-in-the-future) | Pass in an array of NFT metadata and lazy-mint them for others to claim. |

## Implementing It Yourself

This section is meant for advanced users who want to write the functionality from scratch.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@web3sdks/contracts/eip/ERC721A.sol";
import "@web3sdks/contracts/extension/LazyMint.sol";

contract Contract is ERC721A, LazyMint {
    constructor(
        string memory _name,
        string memory _symbol
    )
        ERC721A(
            _name,
            _symbol
        )
    {}

    function _canLazyMint() internal view override returns (bool) {
        // Your custom implementation here
    }
}
```
