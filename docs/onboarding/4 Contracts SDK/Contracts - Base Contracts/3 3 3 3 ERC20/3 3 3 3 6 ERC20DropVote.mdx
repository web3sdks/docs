# ERC20 Drop - Vote

The `ERC20Drop` base contract adds the [Drop](/sdk/interacting-with-contracts/erc20/erc20drop) extension to the [ERC20](/contractkit/base-contracts/erc-20/erc20base)
to allow you to distribute your tokens under the criteria of claim conditions. It also
implements [ERC20Votes](https://github.com/OpenZeppelin/openzeppelin-contracts/blob/master/contracts/token/ERC20/extensions/ERC20Votes.sol),
which provides voting and delegation functionality, along with delegation by signature.

## Implementing the Contract Extension

Import the contract extension and make your contract inherit it.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@web3sdks/contracts/base/ERC20DropVote.sol";

contract MyToken is ERC20DropVote {
	  constructor(
        string memory _name,
        string memory _symbol,
        address _primarySaleRecipient
    )
        ERC20DropVote(
            _name,
            _symbol,
            _primarySaleRecipient
        )
    {}
}
```
