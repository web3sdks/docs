# Base Contracts

<div className="heroGrid">
  
  <div>
    <p>
      Each base smart contract packages together a unique set of smart contract <a href='/contractkit/extensions'>extensions</a> so that you can focus on
      adding your custom logic on top, instead of starting from scratch.
    </p>

Rather than implementing EIP standards manually, base contracts provide a logical foundation for you to extend and customize to your needs.

We provide several base contracts for [ERC721](/contractkit/base-contracts/erc-721/erc721base), [ERC1155](/contractkit/base-contracts/erc-1155/erc1155base), and
[ERC20](/contractkit/base-contracts/erc-20/erc20base); each with different levels of functionality depending on your use case.

You can override and add additional functionality to the base logic, and include [extensions](/contractkit/extensions) to supplement the base functionality.

</div>

## List of Base Contracts

Below is a list of all the base contracts available.

### ERC721

| Name                                                                           | Description                                                                                                                                                                                                                                                                                                                    |
| ------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [ERC721Base](/contractkit/base-contracts/erc-721/erc721base)                   | Implementation of the [ERC721A](https://www.erc721a.org/) standard with the capability to mint NFTs.                                                                                                                                                                                                                           |
| [ERC721SignatureMint](/contractkit/base-contracts/erc-721/erc721signaturemint) | Adds [signature-based minting](/sdk/advanced-features/on-demand-minting) capability to the [ERC721Base](/contractkit/base-contracts/erc-721/erc721base) contract.                                                                                                                                                              |
| [ERC721LazyMint](/contractkit/base-contracts/erc-721/erc721lazymint)           | Adds [lazy minting](/contractkit/extension-contracts/lazymint) capability to the [ERC721Base](/contractkit/base-contracts/erc-721/erc721base). Batch lazy-mint NFT metadata and have other wallets claim them under conditions you define in a `verifyClaim` function.                                                         |
| [ERC721DelayedReveal](/contractkit/base-contracts/erc-721/erc721delayedreveal) | Adds [delayed reveal](/contractkit/extension-contracts/delayedreveal) to the [ERC721LazyMint](/contractkit/base-contracts/erc-721/erc721lazymint) contract.                                                                                                                                                                    |
| [ERC721Drop](/contractkit/base-contracts/erc-721/erc721drop)                   | Adds [delayed reveal](/contractkit/extension-contracts/delayedreveal) and [claim conditions](/contractkit/extension-contracts/dropsinglephase) to the [ERC721LazyMint](/contractkit/base-contracts/erc-721/erc721lazymint) contract. Batch lazy-mint NFTs, and set up claim conditions to define how your NFTs can be claimed. |

### ERC1155

| Name                                                                              | Description                                                                                                                                                                                                                                                                                                                                                                           |
| --------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [ERC1155Base](/contractkit/base-contracts/erc-1155/erc1155base)                   | Implementation of the [ERC1155](https://eips.ethereum.org/EIPS/eip-1155) standard with the capability to mint and burn NFTs.                                                                                                                                                                                                                                                          |
| [ERC1155SignatureMint](/contractkit/base-contracts/erc-1155/erc1155signaturemint) | Adds [signature-based minting](/sdk/advanced-features/on-demand-minting) capability to the [ERC1155Base](/contractkit/base-contracts/erc-1155/erc1155base) contract.                                                                                                                                                                                                                  |
| [ERC1155LazyMint](/contractkit/base-contracts/erc-1155/erc1155lazymint)           | Adds [lazy minting](/contractkit/extension-contracts/lazymint) capability to the [ERC1155Base](/contractkit/base-contracts/erc-1155/erc1155base) contract. Batch lazy-mint NFT metadata and have other wallets claim them under conditions you define in a `verifyClaim` function.                                                                                                    |
| [ERC1155DelayedReveal](/contractkit/base-contracts/erc-1155/erc1155delayedreveal) | Adds [delayed reveal](/contractkit/extension-contracts/delayedreveal) to the [ERC1155Base](/contractkit/base-contracts/erc-1155/erc1155base) contract.                                                                                                                                                                                                                                |
| [ERC1155Drop](/contractkit/base-contracts/erc-1155/erc1155drop)                   | Adds [delayed reveal](/contractkit/extension-contracts/delayedreveal) and [claim conditions](/contractkit/extension-contracts/dropsinglephase1155) to the [ERC1155Base](/contractkit/base-contracts/erc-1155/erc1155base) contract. Batch lazy-mint NFTs, and set up [claim conditions](/contractkit/extension-contracts/dropsinglephase1155) to define how your NFTs can be claimed. |

### ERC20

| Name                                                                                 | Description                                                                                                                                                                                                                                      |
| ------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| [ERC20Base](/contractkit/base-contracts/erc-20/erc20base)                            | Implementation of the [ERC20](https://eips.ethereum.org/EIPS/eip-20) standard with support for the [EIP-2612](https://eips.ethereum.org/EIPS/eip-2612) standard.                                                                                 |
| [ERC20Vote](/contractkit/base-contracts/erc-20/erc20vote)                            | Add [vote](https://github.com/web3sdks/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20Base](/contractkit/base-contracts/erc-20/erc20base) contract.                       |
| [ERC20SignatureMint](/contractkit/base-contracts/erc-20/erc20signaturemint)          | Adds [signature-based minting](/sdk/advanced-features/on-demand-minting) capability to the [ERC20Base](/contractkit/base-contracts/erc-20/erc20base) contract.                                                                                   |
| [ERC20SignatureMintVote](/contractkit/base-contracts/erc-20/erc20signaturemint-vote) | Adds [vote](https://github.com/web3sdks/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20SignatureMint](/contractkit/base-contracts/erc-20/erc20signaturemint) contract.    |
| [ERC20Drop](/contractkit/base-contracts/erc-20/erc20drop)                            | Adds [claim conditions](/contractkit/extension-contracts/dropsinglephase) capability to the [ERC20Base](/contractkit/base-contracts/erc-20/erc20base) contract. Distribute your token for a price under the criteria set up in claim conditions. |
| [ERC20DropVote](/contractkit/base-contracts/erc-20/erc20drop-vote)                   | Adds [vote](https://github.com/web3sdks/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20Drop](/contractkit/base-contracts/erc-20/erc20drop) contract.                      |
